This endpoint merges multiple video files from URLs into a single video.

## Endpoint

`POST /merge-videos/`

**Full URL:** `https://ffmpeg-api-1-hmzg.onrender.com/merge-videos/`

### Form Data Parameters

- `video_urls` (string): Comma-separated list of public video URLs (minimum 2 required)

### Response

```json
{
  "message": "Videos merged successfully",
  "output_file": "https://ffmpeg-api-1-hmzg.onrender.com/uploads/<id>_merged.mp4"
}
```

<Callout type="info">
Ensure all input videos are accessible public URLs and in compatible formats (e.g., MP4).
</Callout>

const formData = new FormData();

fetch("https://ffmpeg-api-1-hmzg.onrender.com/merge-videos/", {

// /app/api/merge-videos.js (Next.js 13+ App Router)
"use client";

export async function mergeVideos() {
  const formData = new FormData();
  formData.append("video_urls", "https://example.com/clip1.mp4, https://example.com/clip2.mp4");

  const res = await fetch("https://ffmpeg-api-1-hmzg.onrender.com/merge-videos/", {
    method: "POST",
    body: formData,
  });

  const data = await res.json();
  console.log("Merged Video:", data.output_file);
}
