This endpoint downloads an image from a given URL and converts it to the specified format using FFmpeg.

## Endpoint

`POST /convert_image_from_url/`

**Full URL:** `https://ffmpeg-api-1-hmzg.onrender.com/convert_image_from_url/`

### Form Data Parameters

* `image_url` (string): The direct URL to the source image.
* `target_format` (string): Desired format to convert to (e.g., `png`, `webp`, `jpg`).

### Success Response

```json
{
  "message": "Image converted to <target_format> successfully.",
  "output_file": "https://ffmpeg-api-1-hmzg.onrender.com/uploads/<uuid>_converted.<target_format>"
}
```

### Error Response

```json
{ "error": "<server error message>" }
```

<Callout type="info">
The image is temporarily downloaded, converted using FFmpeg, and made available via a generated public URL.
</Callout>

## Usage Examples

\<Tabs.Tab>

```bash
curl -X POST https://ffmpeg-api-1-hmzg.onrender.com/convert_image_from_url/ \

  -F "target_format=png"
```

\<Tabs.Tab>

```jsx
const formData = new FormData();
formData.append("image_url", "https://example.com/sample.jpg");
formData.append("target_format", "png");

fetch("https://ffmpeg-api-1-hmzg.onrender.com/convert_image_from_url/", {
  method: "POST",
  body: formData
})
  .then(res => res.json())
  .then(data => console.log(data.output_file));
```

```js
// /app/api/convert-image.js (Next.js 13+ App Router)
"use client";

export async function convertImage() {
  const formData = new FormData();
  formData.append("image_url", "https://example.com/sample.jpg");
  formData.append("target_format", "png");

  const res = await fetch("https://ffmpeg-api-1-hmzg.onrender.com/convert_image_from_url/", {
    method: "POST",
    body: formData,
  });

  const data = await res.json();
  console.log("Converted Image URL:", data.output_file);
}
```
